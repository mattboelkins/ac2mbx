Item 2
The outer function is \(f(x) = \sqrt{x}\), while the inner function is \(g(x) = \tan(x)\), and we know that \[ f'(x) = \frac{1}{2\sqrt{x}}, g'(x) = \sec^2(x), \ \mbox{and} \ f'(g(x)) = \frac{1}{2\sqrt{\tan(x)}}. \] Hence, by the chain rule, \[ h'(x) = f'(g(x))g'(x) = \frac{\sec^2(x)}{2\sqrt{\tan(x)}}. \]
