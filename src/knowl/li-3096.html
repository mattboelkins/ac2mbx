Item 3
Based on the previous part of this problem the \(n\)th order Taylor polynomial for \(\cos(x)\) is \[ 1 - \frac{x^2}{2} + \frac{x^4}{4!} - \frac{x^6}{6!} + \cdots + (-1)^{n/2}\frac{x^n}{n!} \] if \(n\) is even and \[ 1 - \frac{x^2}{2} + \frac{x^4}{4!} - \frac{x^6}{6!} + \cdots + (-1)^{(n-1)/2}\frac{x^(n-1)}{(n-1)!} \] if \(n\) is odd. \end{thmlist}
