Because this limit has indeterminate form \(\frac{\infty}{\infty}\), L'Hopital's Rule tells us that \[ \lim_{x \to \infty} \frac{e^{x} + x}{2e^{x} + x^2} = \lim_{x \to \infty} \frac{e^{x} + 1}{2e^{x} + 2x}. \] The latest limit is indeterminate for the same reason, and a second application of the rule shows \[ \lim_{x \to \infty} \frac{e^{x} + x}{2e^{x} + x^2} = \lim_{x \to \infty} \frac{e^{x}}{2e^{x} + 2}. \] Note how each application of the rule produces a simpler numerator and denominator. With one more use of L'Hopital's Rule, followed by a simple algebraic simplification, we have \[ \lim_{x \to \infty} \frac{e^{x} + x}{2e^{x} + x^2} = \lim_{x \to \infty} \frac{e^{x}}{2e^{x}} = \lim_{x \to \infty} \frac{1}{2} = \frac{1}{2}. \]
