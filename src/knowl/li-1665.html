Item 3
Observe that each term in the sum \[ \ds 3 + 7 + 11 + 15 + \cdots + 27 \] differs from the previous term by 4. If we view \(4\) as \(4 = 4 \cdot 1 - 1\) and \(7\) as \(7 = 4 \cdot 2 - 1\), we see that the pattern may be represented through the function \(f(k) = 4k-1\), so that \[ \ds 3 + 7 + 11 + 15 + \cdots + 27 = \sum_{k=1}^{7} 4k-1. \] We note that \(k=7\) is the end value of the index since \(4 \cdot 7 = 28\).
