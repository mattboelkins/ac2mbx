Since \(C(x) = f(g(x)),\) it follows \(C'(x) = f'(g(x))g'(x).\) Therefore, \(C'(2) = f'(g(2))g'(2)\). From the given table, \(g(2) = -1\), so applying this result and using additional given information, \[ C'(2) = f'(-1) g'(2) = (-5)(2) = -10. \] For \(D(x) = f(f(x))\), the chain rule tells us that \(D'(x) = f'(f(x))f'(x)\), so \(D'(-1) = f'(f(-1))f'(-1)\). Using the given table, it follows \[ D'(-1) = f'(2)f'(-1) = (4)(-5) = -20. \]
