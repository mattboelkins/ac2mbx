\item For all positive real numbers \(x\), \(\ds \frac{d}{dx}[\ln(x)] = \frac{1}{x}\). \item For all real numbers \(x\) such that \(-1 \le x \le 1\), \(\ds \frac{d}{dx}[\arcsin(x)] = \frac{1}{\sqrt{1-x^2}}\). In addition, for all real numbers \(x\), \(\ds \frac{d}{dx}[\arctan(x)] = \frac{1}{1+x^2}\). \item If \(g\) is the inverse of a differentiable function \(f\), then for any point \(x\) in the domain of \(g'\),
\(\ds g'(x) = \frac{1}{f'(g(x))}.\)
