The context of position, velocity, and acceleration is an excellent one in which to understand how a function, its first derivative, and its second derivative are related to one another. In ActivityÂ 1.6.10, we can replace \(s\), \(v\), and \(a\) with an arbitrary function \(f\) and its derivatives \(f'\) and \(f''\), and essentially all the same observations hold. In particular, note that \(f'\) is increasing if and only if both \(f\) is concave up, and similarly \(f'\) is increasing if and only if \(f''\) is positive. Likewise, \(f'\) is decreasing if and only if both \(f\) is concave down, and \(f'\) is decreasing if and only if \(f''\) is negative.
