Solution 1.7.2.1
  1. \(f(-2) = \frac{2}{3}(-2+2) + 1 = 1\); \(f(-1)\) is not defined; \(f(0) = \frac{2}{3}(0+2)+1 = \frac{7}{3}\); \(f(1) = 2\) (by the rule); \(f(2) = 4-2 = 2\).
  2. \[ \lim_{x \to -2^-} f(x) = 2 \ \mbox{and} \lim_{x \to -2^+} f(x) = 1 \]\[ \lim_{x \to -1^-} f(x) = \frac{5}{3} \ \mbox{and} \lim_{x \to -1^+} f(x) = \frac{5}{3} \]\[ \lim_{x \to 0^-} f(x) = \frac{7}{3} \ \mbox{and} \lim_{x \to 0^+} f(x) = \frac{7}{3} \]\[ \lim_{x \to 1^-} f(x) = 3 \ \mbox{and} \lim_{x \to 1^+} f(x) = 3 \]\[ \lim_{x \to 2^-} f(x) = 2 \ \mbox{and} \lim_{x \to 2^+} f(x) = 2 \]
  3. \(\ds \lim_{x \to -2} f(x)\) does not exists because the left-hand limit is \(2\) while the right-hand limit is \(1\). All of the other requested limits exist, as left- and right-hand limits exist and are equal in each case. The respective values of the limits as \(x \to a\) for \(a = -1, 0, 1, 2\) are \(\frac{5}{3}, \frac{7}{3}, 3, 2\).
  4. For \(a = -2\), \(a = -1\), and \(a = 1\), \(\ds \lim_{x \to a} f(x) \ne f(a)\). At \(a = -2\), the limit fails to exist, but \(f(-2) = 1\). At \(a = -1\), the limit is \(\frac{5}{3}\), but \(f(-1)\) is not defined. At \(a = 1\), the limit is 3, but \(f(1) = 2\).
