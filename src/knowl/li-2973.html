Item
Let \(P_2(x) = x - \frac{x^2}{2}\). Show that \(P_2(0) = f(0)\), \(P'_2(0) = f'(0)\), \text{and} \(P''_2(0) = f''(0)\). Use \(P_2(x)\) to approximate \(\ln(2)\) by using the fact that \(P_2(1) \approx f(1)\). Solution

The derivatives of \(P_2\) and \(f\) are

\begin{align*} P_2(x) \amp = x - \frac{x^2}{2} \amp f(x) \amp = \ln(1+x)\\ P'_2(x) \amp = 1-x \amp f'(x) \amp = \frac{1}{1+x}\\ P''_2(x) \amp = -1 \amp f''(x) \amp = -\frac{1}{(1+x)^2}, \end{align*}

and so the derivatives of \(P_2\) and \(f\) evaluated at 0 are

\begin{align*} P_2(0) \amp = 0 \amp f(0) \amp = \ln(1) = 0\\ P'_2(0) \amp = 1 \amp f'(0) \amp = \frac{1}{1+0} = 1\\ P''_2(0) \amp = -1 \amp = -\frac{1}{(1+0)^2} = -1. \end{align*}

Then

\[ \ln(2) = \ln(1+1) \approx P_2(1) = 1 - \frac{1}{2} = \frac{1}{2}. \]
