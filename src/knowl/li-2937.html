Item 3
We use the Ratio Test here with \(a_k = \frac{10^k}{k!}\). Now \begin{align*} \lim_{k \to \infty} \frac{a_{k+1}}{a_k} \amp = \lim_{k \to \infty} \frac{ \frac{10^{k+1}}{(k+1)!} }{ \frac{10^k}{k!} }\\ \amp = \lim_{k \to \infty} \frac{10^{k+1}(k!)}{10^k(k+1)!}\\ \amp = \lim_{k \to \infty} \frac{10}{k+1}\\ \amp = 0, \end{align*} so the Ratio Test shows that \(\ds \sum \frac{10^k}{k!}\) converges.
