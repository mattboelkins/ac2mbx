Solution 4.2.2.1
  1. \begin{align*} \sum_{k=1}^{5} (k^2 + 2) \amp = \amp (1^2 + 2) + (2^2 + 2) + (3^2 + 2) + (4^2 + 2) + (5^2 + 2)\\ \amp = \amp 3 + 6 + 11 + 17 + 27\\ \amp = \amp 64 \end{align*}
  2. \begin{align*} \sum_{i=3}^{6} (2i-1) \amp = \amp (2 \cdot 3 - 1) + (2 \cdot 4- 1) + (2 \cdot 5 - 1) + (2 \cdot 6 - 1)\\ \amp = \amp 5 + 7 + 9 + 11\\ \amp = \amp 32 \end{align*}
  3. Observe that each term in the sum \[ \ds 3 + 7 + 11 + 15 + \cdots + 27 \] differs from the previous term by 4. If we view \(4\) as \(4 = 4 \cdot 1 - 1\) and \(7\) as \(7 = 4 \cdot 2 - 1\), we see that the pattern may be represented through the function \(f(k) = 4k-1\), so that \[ \ds 3 + 7 + 11 + 15 + \cdots + 27 = \sum_{k=1}^{7} 4k-1. \] We note that \(k=7\) is the end value of the index since \(4 \cdot 7 = 28\).
  4. The sum \(\ds 4 + 8 + 16 + 32 + \cdots + 256\) is a sum of powers of \(2\), which we can express in sigma notation as \[ 4 + 8 + 16 + 32 + \cdots + 256 = \sum{i=2}^{8} 2^i. \]
  5. \begin{align*} \sum_{i=1}^{6} \frac{1}{2^i} \amp = \amp \frac{1}{2} + \frac{1}{2^2} + \cdots + \frac{1}{2^6}\\ \amp = \amp \frac{32}{64} + \frac{16}{64} + \frac{8}{64} + \frac{4}{64} + \frac{2}{64} + \frac{1}{64}\\ \amp = \amp \frac{63}{64}. \end{align*}
