Hint 2.8.2.2
  1. Remember that \(\ln(1) = 0\), so this limit takes on the indeterminate form \(\frac{0}{0}\).
  2. Note that \(x \to \pi\), not \(x \to 0\), so this limit is not indeterminate. The denominator tends to \(\pi\); what does the numerator approach?
  3. Observe that \(e^{x-1} \to 1\) as \(x \to 1\), so this limit is indeterminate in the form \(\frac{0}{0}\).
  4. After one application of L'Hopital's Rule, you should find yet another indeterminate form, to which you can again apply the rule.
