Hint 5.1.2.2
  1. Consider the sign of \(F' = f\) and recall that wherever \(F' > 0\), \(F\) is increasing.
  2. Consider the sign of \(F'' = f'\) and recall that wherever \(F''>0\), \(F\) is concave up. Note particularly that \(F''>0\) if and only if \(f\) is increasing.
  3. Where does \(F' = f\) change sign? A relative maximum for \(F\) will occur wherever \(F'\) changes from positive to negative.
  4. Recall that \(F(1) = F(0) + \int_0^1 f(t) \, dt\). Similarly, \(F(2) = F(0) + \int_0^1 f(t) \, dt\).
  5. Use the function values found in (d) and the earlier information regarding the shape of \(F\).
  6. Note that \(G(1) = G(0) + \int_0^1 f(t) \, dt\). Since \(G(0) = 0\) (while \(F(0) = -1\)), this changes each response in (e) by the same constant amount.
