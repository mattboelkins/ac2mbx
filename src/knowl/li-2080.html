Item 1
For each of the following functions, use the Product Rule to find the function's derivative. Be sure to label each derivative by name (e.g., the derivative of \(g(x)\) should be labeled \(g'(x)\)).
  1. \(g(x) = x\sin(x)\)
  2. \(h(x) = xe^x\)
  3. \(p(x) = x\ln(x)\)
  4. \(q(x) = x^2 \cos(x)\)
  5. \(r(x) = e^x \sin(x)\)
