Hint 2.2.2.4
  1. It's very important to use a straightedge for accuracy.
  2. \(g'(\frac{\pi}{2}) \approx -1\), as for each grid box we move horizontally, we go down about 1.5 grid boxes.
  3. \(g'(\frac{\pi}{2}) \approx \frac{\cos(\frac{\pi}{2}+h)}{h}\) for small values of \(h\). Try using \(h = 0.001\) and \(h = -.001\).
  4. Recall that heights on \(g'\) come from slopes on \(g\). From your earlier work, you should know several places \(g'(x) = 0\), plus that \(g'(0) \approx 1\). Plot these values on the grid for \(y = g'(x)\).
  5. It might be reasonable to expect that the derivative of a trigonometric function is another trigonometric function, or possibly some multiple of such a function.
