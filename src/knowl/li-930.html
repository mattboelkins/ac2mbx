Item 5
The outer function is \(f(x) = x^9\), while the inner function is \(g(x) = \sec(x) + e^x\), and we know that \[ f'(x) = 9x^8, g'(x) = \sec(x)\tan(x) + e^x, \ \mbox{and} \ f'(g(x)) = 9(\sec(x)+e^x)^8. \] Hence, by the chain rule, \[ h'(x) = f'(g(x))g'(x) = 9(\sec(x)+e^x)^8 (\sec(x)\tan(x) + e^x). \]
