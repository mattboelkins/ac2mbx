Solution 4.4.3.1

\vfill

  1. By standard antiderivative rules and the FTC, \begin{align*} \int_0^1 \left(x^3 - x - e^x + 2\right) \,dx \amp = \amp \left. \frac{1}{4} x^4 - \frac{1}{2}x^2 - e^x + 2x \right|_0^1\\ \amp = \amp \left(\frac{1}{4} (1)^4 - \frac{1}{2}(1)^2 - e^1 + 2(1) \right) - \left(\frac{1}{4} (0)^4 - \frac{1}{2}(0)^2 - e^0 + 2(0) \right)\\ \amp = \amp \frac{1}{4} - \frac{1}{2} - e + 2 - \left(0 - 0 - 1 + 0 \right)\\ \amp = \amp -\frac{1}{4} - e + 3\\ \amp = \amp \frac{11}{4} - 3. \end{align*}
  2. Calculating the needed antiderivative and applying the FTC, \begin{align*} \int_0^{\pi/3} (2\sin (t) - 4\cos(t) + \sec^2(t) - \pi) \, dt \amp = \amp \left. \left(-2\cos(t) - 4\sin(t) + \tan(t) - \pi t \right) \right|_0^{\pi/3}\\ \amp = \amp \left(-2\cos(\pi/3) - 4\sin(\pi/3) + \tan(\pi/3) - \pi (\pi/3) \right) -\\ \amp \ \amp \left(-2\cos(0)) - 4\sin(0) + \tan(0) - \pi (0) \right)\\ \amp = \amp -2 \cdot \frac{1}{2} - 4 \cdot \frac{\sqrt{3}}{2} + \sqrt{3} - \frac{\pi^2}{3} - (-2 - 0 + 0 - 0)\\ \amp = \amp 1 - \sqrt{3} - \frac{\pi^2}{3}. \end{align*}
  3. Noting that \(\frac{d}{dx}[\frac{2}{3} x^{3/2}] = x^{1/2}\), we find that \begin{align*} \int_0^1 (\sqrt{x} - x^2) \, dx \amp = \amp \left. \frac{2}{3}x^{3/2} - \frac{1}{3}x^3 \right|_0^1\\ \amp = \amp \frac{2}{3} - \frac{1}{3} - (0 - 0 )\\ \amp = \amp \frac{1}{3}. \end{align*}
