Solution 1.4.2.2
  1. \(f'(x) = 0\) because the slope of the tangent line to the horizontal line given by \(f(x) = 1\) is zero at every value of \(x\).
  2. \(g'(t) = 1\) because the slope of the tangent line to the line given by \(g(t) = t\) is 1 at every value of \(t\).
  3. \begin{align*} p'(z) \amp = \amp \lim_{h \to 0} \frac{p(z+h) - p(z)}{h}\\ \amp = \amp \lim_{h \to 0} \frac{(z+h)^2 - z^2}{h}\\ \amp = \amp \lim_{h \to 0} \frac{z^2 + 2zh + h^2 - z^2}{h}\\ \amp = \amp \lim_{h \to 0} \frac{2zh + h^2}{h}\\ \amp = \amp \lim_{h \to 0} 2z + h\\ \amp = \amp 2z \end{align*}
  4. \begin{align*} q'(s) \amp = \amp \lim_{h \to 0} \frac{q(s+h) - q(s)}{h}\\ \amp = \amp \lim_{h \to 0} \frac{(s+h)^3 - s^3}{h}\\ \amp = \amp \lim_{h \to 0} \frac{s^3 + 3s^2h + 3sh^2 + h^3 - s^3}{h}\\ \amp = \amp \lim_{h \to 0} \frac{3s^2h + 3sh^2 + h^2}{h}\\ \amp = \amp \lim_{h \to 0} 3s^2 + 3sh + h\\ \amp = \amp 3s^2 \end{align*}
  5. \begin{align*} F'(t) \amp = \amp \lim_{h \to 0} \frac{F(t+h) - F(s)}{h}\\ \amp = \amp \lim_{h \to 0} \frac{\frac{1}{t+h} - \frac{1}{t}}{h}\\ \amp = \amp \lim_{h \to 0} \frac{\frac{t - (t+h)}{t(t+h)}}{h}\\ \amp = \amp \lim_{h \to 0} \frac{-h}{ht(t+h)}\\ \amp = \amp \lim_{h \to 0} \frac{-1}{t(t+h)}\\ \amp = \amp \frac{-1}{t^2} \end{align*}
  6. \begin{align*} G'(y) \amp = \amp \lim_{h \to 0} \frac{G(y+h) - G(y)}{h}\\ \amp = \amp \lim_{h \to 0} \frac{\sqrt{y+h}-\sqrt{y}}{h}\\ \amp = \amp \lim_{h \to 0} \frac{\sqrt{y+h}-\sqrt{y}}{h} \cdot \frac{\sqrt{y+h} + \sqrt{y}}{\sqrt{y+h} + \sqrt{y}}\\ \amp = \amp \lim_{h \to 0} \frac{(y+h)-y}{h(\sqrt{y+h} + \sqrt{y})}\\ \amp = \amp \lim_{h \to 0} \frac{h}{h(\sqrt{y+h} + \sqrt{y})}\\ \amp = \amp \lim_{h \to 0} \frac{1}{\sqrt{y+h} + \sqrt{y}}\\ \amp = \amp \frac{1}{2\sqrt{y}} \end{align*}
