Hint 4.2.2.2
  1. Observe that when \(k = 1\), \(k^2 + 2 = 1^2 + 2 = 3\). This is the first term in the sum. The last term in the sum occurs when \(k = 5\), which is \(5^2 + 2 = 27\).
  2. Note that this sum starts at \(i = 3\) and the first term is \((2\cdot 3 - 1) = 5\).
  3. Since the terms in the sum increase by 4, try a function \(f(k)\) that somehow involves \(4k\). You have freedom to decide where your index \(k\) starts; try \(k = 1\).
  4. What pattern do you observe in the terms of the sum?
  5. Write every term in the sum as a fraction with denominator \(2^6 = 64\).
