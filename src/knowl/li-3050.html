Item 7
This series looks geometric, but to be sure let's write out the first few terms of this series: \[ \sum_{k=2}^{\infty} \ \frac{3^{k-1}}{7^k} = \frac{3}{7^2} + \frac{3^2}{7^3} + \frac{3^3}{7^4} + \cdots = \left( \frac{3}{7^2} \right) \left[1 + \frac{3}{7} + \left( \frac{3}{7} \right)^2 + \left( \frac{3}{7} \right)^3 + \cdots \right]. \] So \(\ds \sum_{k=2}^{\infty} \ \frac{3^{k-1}}{7^k}\) is a geometric series with \(a = \frac{3}{49}\) and \(r = \frac{3}{7}\). Since the ratio of this geometric series is between \(-1\) and 1, the series converges. The sum of the series is \begin{align*} \sum_{k=2}^{\infty} \ \frac{3^{k-1}}{7^k} \amp = \left(\frac{3}{49}\right) \left(\frac{1}{1-\frac{3}{7}} \right)\\ \amp = \left(\frac{3}{49}\right)\left(\frac{7}{4}\right)\\ \amp = \frac{3}{28}. \end{align*}
