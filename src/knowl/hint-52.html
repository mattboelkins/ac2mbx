Hint 2.2.2.2
  1. It's very important to use a straightedge for accuracy.
  2. \(f'(0) \approx 1\), as for each grid box we move horizontally, we go up about 1.5 grid boxes.
  3. \(f'(0) \approx \frac{\sin(h)}{h}\) for small values of \(h\). Try using \(h = 0.001\) and \(h = -.001\).
  4. Recall that heights on \(f'\) come from slopes on \(f\). From your earlier work, you should know several places \(f'(x) = 0\), plus that \(f'(0) \approx 1\). Plot these values on the grid for \(y = f'(x)\).
  5. It might be reasonable to expect that the derivative of a trigonometric function is another trigonometric function, or possibly some multiple of such a function.
