Item 3
With \(q(x) = \frac{x^2}{x-2}\), we have \[ q'(x) = \frac{(x-2)(2x) - x^2(1)}{(x-2)^2} = \frac{2x^2 - 4x - x^2}{(x-2)^2} = \frac{x^2-4x}{(x-2)^2} = \frac{x(x-4)}{(x-2)^2}. \] Hence, the critical values of \(q\) are \(x = 0\) and \(x = 4\). Only the latter critical value lies in the interval \([3,7]\), and thus we evaluate \(q\) and find
  • \(q(4) = \frac{16}{2} = 8\)
  • \(q(3) = \frac{9}{1} = 9\)
  • \(q(7) = \frac{49}{5} = 9.8\)
We now see that on \([3,7]\) the absolute maximum of \(q\) is 9.8 and the absolute minimum is 8.
