Hint 2.1.4.2
  1. Use the sum rule; think about which functions in the sum are power functions and which are exponential.
  2. Use the sum rule together with the constant multiple rule. Note which functions are exponential and which are powers of \(x\).
  3. Recall that \(\sqrt{z} = z^{1/2}\); also rewrite \(\frac{1}{z^4}\) as a power of \(z\).
  4. Note that \(\sqrt{53}\), \(\pi\), and \(e^4\) are all constants.
  5. Expand the product and combine like terms before attempting to find the derivative.
  6. Rewrite the single fraction as a sum of three fractions, and simplify. That is, note that \(q(x) = \ds \frac{x^3 - x + 2}{x} = \ds \frac{x^3}{x} - \frac{x}{x} + \frac{2}{x}\)
  7. Note that “\(a\)” is the independent variable and \(p(a)\) is simply a polynomial in \(a\).
