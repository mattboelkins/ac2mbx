Item 3
Show that \(P_2 = \left(1 + \frac{r}{12}\right)^2P - \left[1 + \left(1+\frac{r}{12}\right)\right] M\). Solution

\vs

We know \(P_2 = \left(1 + \frac{r}{12}\right)P_1 - M\), so we substitute for \(P_1\) and simplify:

\begin{align*} P_2 \amp = \left(1 + \frac{r}{12}\right)P_1 - M\\ \amp = \left(1 + \frac{r}{12}\right)\left[\left(1+\frac{r}{12}\right)P - M\right] - M\\ \amp = \left(1 + \frac{r}{12}\right)^2P - \left[1 + \left(1+\frac{r}{12}\right)\right] M. \end{align*}
