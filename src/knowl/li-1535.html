Item 5
Recalling that \(2h \frac{dh}{dt} = 2z \frac{dz}{dt}\), it follows that \[ \frac{dz}{dt} = \frac{h}{z} \frac{dh}{dt}. \] Using this equation when \(h = 3000\) and \(\frac{dh}{dt} = 600\) led us to conclude that \(\left. \frac{dz}{dt} \right|_{h=3000} = \frac{3}{5} \cdot 600 = 360 \ \mbox{feet/sec} \). If we instead use \(h = 4000\), it follows that \(z = 4000\sqrt{2}\), so that \[ \left. \frac{dz}{dt} \right|_{h=4000} = \frac{4}{4\sqrt{2}} \cdot 600 \approx 424.26 \ \mbox{feet/sec} . \] Indeed, \(\frac{dz}{dt}\) is an increasing function of \(h\), provided that \(\frac{dh}{dt}\) is constant, because we can write \(h^2 + 4000^2 = z^2\), so \(z = \sqrt{h^2 + 4000^2}\), making \[ \frac{dz}{dt} = \frac{h}{\sqrt{h^2 + 4000^2}} \frac{dh}{dt} = \frac{h}{\sqrt{h^2 + 4000^2}} 600. \] It is straightforward to verify that \(\frac{h}{\sqrt{h^2 + 4000^2}}\) is an increasing function of \(h\).
