  3. From the limit definition, \begin{align*} f'(0) \amp = \amp \lim_{h \to 0} \frac{f(0 + h) - f(0)}{h}\\ \amp = \amp \lim_{h \to 0} \frac{\sin(0 + h) - \sin(0)}{h}\\ \amp = \amp \lim_{h \to 0} \frac{\sin(h)}{h} \end{align*} Because we cannot simplify the fraction \(\frac{\sin(h)}{h}\) any further algebraically, we estimate the value of the limit using small values of \(h\). Doing so, it appears that \(\lim_{h \to 0} \frac{\sin(h)}{h} = 1\), and thus \(f'(0) = 1\). This matches the estimate generated visually by sketching the tangent line at \((0,f(0))\). Finally, by the periodicity of the sine function, we expect the value of the derivative at 0 to match the derivative value at \(-2\pi\) and \(2\pi\).
