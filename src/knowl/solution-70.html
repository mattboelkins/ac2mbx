Solution 4.3.3.1
  1. Note that the value of \(\int_2^5 f(x) \, dx\) is given, and thus \[ \int_5^2 f(x) \,dx = -\int_2^5 f(x) \, dx = -2. \]
  2. Since \(\int_0^2 g(x) \,dx = 4\) and \(\int_2^5 g(x) \,dx = -1\), we have \[ \int_0^5 g(x) \,dx = \int_0^2 g(x) \,dx + \int_2^5 g(x) \,dx = 4 + (-1) = 3. \]
  3. First, using work from and similar to that in (c), we find \(\int_0^5 f(x) \, dx = -3 + 2 = -1\) and \(\int_0^5 g(x) \, dx = 3\), thus by the sum rule, \[ \int_0^5 (f(x) + g(x))\, dx = \int_0^5 f(x)\, dx + \int_0^5 g(x)\, dx = -1 + 3 = 2. \]
  4. By the sum and constant multiple rules, \[ \int_2^5 (3x^2 - 4x^3) \, dx = 3\int_2^5 x^2 \, dx - 4\int_2^5 x^3 \, dx = 3 \cdot \frac{8}{3} - 4 \frac{609}{4} = 8 - 609 = -608. \]
  5. First, we write \(\int_5^0 (2x^3 - 7g(x)) \, dx = -\int_0^5 (2x^3 - 7g(x)) \, dx\). Then, using the sum and constant multiple rules, it follows \begin{align*} \int_5^0 (2x^3 - 7g(x)) \, dx \amp = \amp -\int_0^5 (2x^3 - 7g(x)) \, dx\\ \amp = \amp -\left(2 \int_0^5 x^3 \, dx - 7 \int_0^5 g(x) \,dx \right)\\ \amp = \amp -2 \left(\frac{8}{3} + \frac{117}{3}\right) + 7 \left(4 + (-1))\right)\\ \amp = \amp -\frac{250}{3} + 21\\ \amp = \amp -\frac{187}{3}. \end{align*}
