Exercise 5.5.1

For each of the following problems, evaluate the integral by using the partial fraction decomposition provided.

  1. \(\ds \int \frac{1}{x^2 - 2x - 3} \, dx\), \hfill given that \(\frac{1}{x^2 - 2x - 3} = \frac{1/4}{x-3} - \frac{1/4}{x+1}\)
  2. \(\ds \int \frac{x^2+1}{x^3 - x^2} \, dx\), \hfill given that \(\frac{x^2+1}{x^3 - x^2} = -\frac{1}{x} - \frac{1}{x^2} + \frac{2}{x-1}\)
  3. \(\ds \int \frac{x-2}{x^4 + x^2}\, dx\), \hfill given that \(\frac{x-2}{x^4 + x^2} = \frac{1}{x} - \frac{2}{x^2} + \frac{-x+2}{1+x^2}\)
