Item 3
Let \(c\) be a nonnegative number. The derivatives of \(e^x\) are all equal to \(e^x\). Now \(e^x\) is an increasing function so \(e^x \lt e^c\) for all \(x\) in the interval \([0,c]\). So if \(P_n(x)\) is the \(n\)th order Taylor polynomial centered at 0 for \(e^x\), then for any \(x\) in \([0,c]\) we have \[ \left|P_n(x) - f(x)\right| \leq 1\frac{e^c|x|^{n+1}}{(n+1)!}. \] Since \[ \lim_{n \to \infty} e^c\frac{|x|^{n+1}}{(n+1)!} = e^c\lim_{n \to \infty} \frac{|x|^{n+1}}{(n+1)!} = 0, \] we conclude that \(P_n(x) \to f(x)\) as \(n \to \infty\) for all nonnegative \(x\).
