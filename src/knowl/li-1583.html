Item 3
By thinking about how the power rule for differentiation works, we can undo this rule and find a position function \(s\) whose derivative is \(v\). For instance, since \(\frac{d}{dt}[t^4] = 4t^3\), we see that \(\frac{d}{dt}[\frac{1}{8}t^4] = \frac{1}{2}t^3\). Thus, if we let \[ s(t) = \frac{1}{8}t^4 - \frac{1}{2} t^3 + \frac{3}{4} t^2 + \frac{3}{2}t, \] then it is straightforward to check that \(s'(t) = \frac{1}{2}t^3 - \frac{3}{2}t^2 + \frac{3}{2}t + \frac{3}{2}\), which is precisely the formula for \(v(t)\) that we were given.
