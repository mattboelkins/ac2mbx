Solution 4.4.2.1
  1. Because \(\frac{d}{dx}[2x - x^2] = 2-2x\), by the Fundamental Theorem of Calculus, \[ \int_{-1}^4 (2-2x) \, dx = \left. 2x - x^2 \right|_{-1}^4, \] and therefore \[ \int_{-1}^4 (2-2x) \, dx = (2 \cdot 4 - 4^2) - (2(-1) - (-1)^2) = -8 + 3 = -5. \]
  2. Since \(\frac{d}{dx} [\cos(x)] = -\sin(x)\), an antiderivative of \(f(x) = \sin(x)\) is \(F(x) = -\cos(x)\). Therefore, by the FTC, \[ \ds \int_{0}^{\frac{\pi}{2}} \sin(x) \, dx = \left. -\cos(x) \right|_{-\frac{\pi}{2}}^{\frac{\pi}{2}}, \] so \[ \ds \int_{0}^{\frac{\pi}{2}} \sin(x) \, dx = -\cos(\frac{\pi}{2}) - (-\cos(0)) = -0 + 1 = 1. \]
  3. Since \(e^x\) is its own derivative, it is also its own antiderivative. Hence, \[ \int_0^1 e^x \, dx = \left. e^x \right|_0^1 = e^1 - e^0 = e-1. \]
  4. Note that since \(\frac{d}{dx} [x^6] = 6x^5\), it follows that \(\frac{d}{dx} [\frac{1}{6}x^6] = x^5\), and thus \[ \ds \int_{-1}^{1} x^5 \, dx = \left. \frac{1}{6}x^6 \right|_{-1}^1 = \frac{1}{6} (1)^6 - \frac{1}{6}(-1)^6 = 0. \]
  5. Using the sum and constant multiple rules for differentiation, we can see that similar results hold for antidifferentiation, and thus that \(F(x) = \frac{3}{4} x^4 - \frac{2}{3} x^3 - e^x\) is an antiderivative of \(f(x) = 3x^3 - 2x^2 - e^x\). Now, by the FTC, \begin{align*} \int_0^2 (3x^3 - 2x^2 - e^x) \, dx \amp = \amp \left. \frac{3}{4} x^4 - \frac{2}{3} x^3 - e^x \right|_0^2\\ \amp = \amp \frac{3}{4} (2)^4 - \frac{2}{3} (2)^3 - e^2 - (\frac{3}{4} (0)^4 - \frac{2}{3} (0)^3 - e^0)\\ \amp = \amp 12 - \frac{16}{3} - e^2 - (0 - 0 - 1)\\ \amp = \amp \frac{17}{3} - e^2. \end{align*}
