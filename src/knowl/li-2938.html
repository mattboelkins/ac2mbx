Item 4
If we ignore the highest powered terms, then the series \(\ds \sum \frac{k^3-2k^2+1}{k^6+4}\) looks like the series \(\ds \sum \frac{k^3}{k^6} = \sum \frac{1}{k^3}\). We formally apply the Limit Comparison Test as follows: \begin{align*} \lim_{k \to \infty} \frac{ \frac{k^3-2k^2+1}{k^6+4} }{ \frac{1}{k^3} } \amp = \lim_{k \to \infty} \frac{k^6-2k^5+k^3}{k^6+4}\\ \amp = \lim_{k \to \infty} \frac{1 - \frac{2}{k} + \frac{1}{k^3}}{1 + \frac{4}{k^6}}\\ \amp 1. \end{align*} Since \(\ds \sum \frac{1}{k^3}\) is a \(p\)-series with \(p=3>1\), the series \(\ds \sum \frac{1}{k^3}\) converges. Consequently, the Limit Comparison Test shows that \(\ds \sum \frac{k^3-2k^2+1}{k^6+4}\) converges as well.
