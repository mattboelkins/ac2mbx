Exercise 7.4.5

Solve each of the following differential equations or initial value problems.

  1. \(\ds \frac{dy}{dt} - (2-t) y = 2-t\)
  2. \(\ds \frac{1}{t}\frac{dy}{dt} = e^{t^2-2y}\)
  3. \(y' = 2y+2\),  \(y(0)=2\)
  4. \(y' = 2y^2\),  \(y(-1) = 2\)
  5. \(\ds \frac{dy}{dt} = \frac{-2ty}{t^2 + 1}\),  \(y(0) = 4\)
